# ⏱ Трекер рабочего времени с автоматическим отчётом и Email-рассылкой

Учебный проект по дисциплине «Проектная деятельность»

---

##  Описание
Простая система для учёта времени, затраченного на задачи.  
**Ключевые особенности:** 
- Автоматическая генерация отчётов по расписанию
- Отправка отчётов на Email
- Красивые Excel-отчёты с форматированием
---

##  Возможности

### Основные функции:
- Запуск и остановка таймера работы
- Сохранение названия проекта и описания задачи
- Просмотр истории записей в таблице
- Ручной экспорт отчёта в Excel

### Автоматизация:
- Автоматическая генерация отчётов (ежедневно в 09:00 и 18:00)
- Фильтрация по дате (автоматический отчёт — только за сегодня)
- Email-рассылка (отчёт отправляется на почту после генерации)
- Подсчёт общей статистики (часы, проекты, записи)

##  Технологии

| Компонент | Технология | Назначение |
|-----------|-----------|------------|
| Язык | Python 3.14 | Основной язык программирования |
| Веб-фреймворк | Flask 3.0 | Серверная часть, API |
| База данных | SQLite | Хранение записей (файл .db) |
| Планировщик | schedule | Автоматизация задач |
| Email | smtplib | Отправка отчётов на почту |
| Excel | openpyxl | Генерация красивых отчётов |
| Фронтенд | HTML + CSS + JS | Веб-интерфейс |

---

###  Как запустить

## Требования:
- Python 3.8 или выше
- pip (менеджер пакетов Python)

## Установка:
- Откройте терминал в папке backend
- Введите: cd backend
- Введите: pip install -r requirements.txt
- Введите: python app.py
- Откройте браузер: http://localhost:5000

#####  Автоматизация
## Планировщик задач:
Работает в фоновом потоке (не блокирует сервер)
Генерирует отчёты ежедневно в 09:00 и 18:00
Фильтрует записи по дате (только за текущий день)
Сохраняет отчёты в папку backend/reports/auto/

## Email-рассылка:
Отправляет отчёт сразу после генерации
Использует SMTP-сервер (Gmail/Яндекс)
Пароль приложения (безопасное хранение)
Вложение: Excel-файл с деталями
